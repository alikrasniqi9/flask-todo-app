# 📝 Flask Todo App (with Docker)

A simple and clean Todo web application built using **Flask**, **SQLite**, and **Docker**.  
Supports full CRUD operations (Create, Read, Update, Delete).

---

## 🚀 Features

- Add new tasks
- Edit existing tasks
- Delete tasks
- View all tasks
- SQLite database integration
- Docker support for containerization

---

## 🛠️ Tech Stack

- Python 3.14
- Flask
- Flask-SQLAlchemy
- SQLite
- Docker
